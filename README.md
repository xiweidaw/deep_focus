# 🧘 DeepFocus (静笃)

> "知止而后有定，定而后能静。" —— 极简主义心流效率工具。

DeepFocus 是一款拒绝繁杂、专注于当下的生产力工具。它结合了倒计时、正计时与日记复盘系统，旨在帮助用户在碎片化时代重获深度专注的能力。

## ✨ 核心特性 (Features)

- **极简美学**: 全局宋体排版，黑金/纸白双色主题，极致的留白与呼吸感设计。
- **多模式计时**: 支持标准番茄钟倒计时、深度工作正计时（Flow Mode）。
- **防误触机制**: 独特的长按圆环退出交互，防止无意中断心流。
- **触觉反馈**: 精细打磨的 Haptic Feedback，让每一次开始与结束都有质感。
- **日记复盘**: 专注结束后的无压力记录逻辑，自动生成每日数据卡片与时间轴。
- **隐私优先**: 基于 Hive 本地数据库，数据完全离线存储，安全无忧。

## 🛠 技术栈 (Tech Stack)

- **Framework**: Flutter (Dart)
- **Architecture**: Clean Architecture + Provider
- **Local DB**: Hive (NoSQL)
- **UI/UX**: Custom Painters, Animations, Dynamic Theming
